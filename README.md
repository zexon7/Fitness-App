# Fitness-App
    使用Android Studio與Firebase製作的健生App
# 功能
  * 登入
  * 記錄當日食物消耗
  * 使用者資料記錄
  * 未完善功能
    * 日期切換無法顯示食物消耗記錄
    * Weight分頁圖表記錄
    * 卡路里消耗計算
